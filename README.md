# Refactoring plugin for PhpStorm

# How to install?
1. Download latest version from [github](https://github.com/funivan/PhpStorm-Refactoring-Plugin/releases)
2. Invoke action in PHPStorm `Install plugin from disk`
3. Restart PHPstorm
